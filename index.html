Descendant Combinator [define with space]

It matches all the descedant elements of the specified element

Child Combinator [>]

It selects all the immediate children of the specified element

Adjacent Sibling [+]

Selects immediately following sibling of the specified element

General Sibling [~]

This selects all the siblings after specified element
